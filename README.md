# HibiscusPlus Premium Website

A luxury Next.js + Tailwind website for HibiscusPlus Limited, ready for Vercel deployment.

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy to Vercel
1. Upload this folder to a new GitHub repository.
2. In Vercel, click **Add New → Project**.
3. Import the GitHub repository.
4. Click **Deploy**.

## Connect your domain
After deployment, add `hibiscusplus.co.uk` in Vercel Project → Settings → Domains.
For IONOS DNS, use:
- A record: `@` → `76.76.21.21`
- CNAME: `www` → `cname.vercel-dns.com`
